Yersinia Pestis Genome Assembly Code

Assembly Code

# ABySS
# Activate mamba genome analysis environment
mamba activate genomeassembly
# Navigate into the folder with your sequencing reads
cd /mnt/c/Users/eh040/Desktop/Bioinformatics/FinalProject/ypestis_reads
# Running ABySS (pe=paired end reads; name=prefix name of output files; k=k-mer; B=memory amount; in=input)
abyss-pe name=ypestis k=96 B=2G in='ypestisreads1.fastq ypestisreads2.fastq'

# SPAdes
# Go to https://usegalaxy.eu/
# Upload Data under 'Tools'
# Upload the two reads files
# Navigate to the SPAdes genome assembler
# Choose 'Paired-ends: individual data sets'
# Select the first reads file for the forward reads and the second reads file for the reverse reads
# RunTool

# Quast
# Download reference genome (.fna) and reference annotation (.gff) from NCBI
# Upload to Galaxy
# Choose Quast Genome assembly Quality under tools
# Choose 'Individual assembly (1 contig file per sample)' under 'Assembly mode?'
# Choose the SPAdes Scaffolds file under 'Contigs/scaffolds file'
# Select your reference genome under 'Reference Genome'
# Select your reference annotation under 'Genomic feature positions in the reference genome' and 'Operon positions in the reference genome'
# Select 'yes' for scaffolds rather than contigs
# Select 'GeneMarkS' under 'Tool for gene prediction'
# Select 'yes' for 'Enables ribosomal RNA gene finding'
# Select 'yes' for 'Enables search for Universal Single-Copy Orthologs using BUSCO'
# Do the same for the ABySS output, but use the 'ypestis-8.fa' file from ABySS

# Prokka
# Activate genome assembly environment
mamba activate genomeassembly
# Navigate into the genome assembly folder you are working in
cd /mnt/c/Users/eh040/Desktop/Bioinformatics/FinalProject
# Run Prokka ('prokka'=calls the program; '--outdir ProkkaAnnotation'=creates an output directory that is called 'ProkkaAnnotation'; '--prefix ypestis'=gives the files a prefix name 'ypestis'; 'ABySSOutput/ypestis-8.fa'=tells the program which file we want it to annotate in the ABySSOutput folder)
prokka --outdir ProkkaAnnotation --prefix ypestis ABySSOutput/ypestis-8.fa
